# South Gippsland Shire Council Scraper

### Setup scraper

```bash
bundle install
```

### Run Localy
```bash
bundle exec ruby scraper.rb
```

### Site Information

This site is a typical .NET site involves the followings.
* Cookie - Yes
* Pagnation - Not yet supported
* Javascript - No
* Ajax - No
* JSON - No
* Structured HTML - Reasonable
